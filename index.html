Command Centre Revamp

High-fidelity interactive prototype -- April 2026

12 screens | 4 workflows | Vanilla HTML + Tailwind + JS | All data is mock/demo

Inbox & Triage

-- 3-pane layout: conversation list + chat thread + context panel
Compact List

44px rows, fast scanning. Best for high-volume triage -- click any conversation to load thread + context panel.
Card View

Rich cards with SLA progress bars, last message preview, and quick action buttons. Context-first browsing.
Queue View

Swim lanes by SLA urgency: Unassigned, Needs Response, Warning, On Track. Manager triage view.

Client Management

-- Individual clients + household grouping
Client List

Full CRM table with toggle between All Clients (12 rows) and Households (6 rows). Click any row for 360 detail.
Client 360 Detail

6 tabs: Overview, Trips, Conversations, Preferences, Points, Activity. Household members sidebar.
Client Onboarding

Progressive disclosure form. Paste a call transcript and AI extracts client preferences automatically.

Trip Management

-- 7-stage pipeline from Lead to Archived
Trip Pipeline (Kanban)

7 columns: Lead, Discovery, Planning, Proposal, Booking, Active, Archived. Filter by agent and date range.
Trip Detail

5 tabs: Overview (AI summary + itinerary), Bookings (component cards), Travelers, Tasks, Activity log.

Operations

-- Manager overview, tasks, alerts, configuration
Dashboard

7 KPI cards, SLA health bar, pipeline summary, team workload table, recent activity feed. Manager home screen.
Task List

Priority-striped table with stats bar. Click any row to open slide-over detail panel with activity log.
Flight Alerts

Alert monitoring table: route, cabin class, points threshold, trigger history. Points-based award flight tracking.
Settings

5 tabs: Team management, SLA configuration, Channels, Appearance (dark/light), Keyboard shortcuts.

SLA monitoring
