Inbox & Triage

All three views share the same 3-pane layout (conversation list + chat thread + context panel with AI copilot). The difference is how the conversation list is presented — each view is designed for a different agent workflow and team role.

Compact List
For: experienced agents, high-volume days

Dense 44px rows — no message preview, maximum conversations visible at once. For agents who know their clients and need to move fast through a full queue.
Card View
For: agents handling complex, high-touch trips

Rich cards with last message preview, SLA progress bar, and inline quick actions. For agents managing fewer but more complex conversations where context before responding matters.
Queue View
For: managers, senior agents doing SLA triage

Swim lanes by urgency: Unassigned → Needs Response → Warning → On Track. Most at-risk conversations surface to the top automatically. Built for shift handoffs and SLA oversight.

Client Management

-- Individual clients + household grouping
Client List

Full CRM table with toggle between All Clients (12 rows) and Households (6 rows). Click any row for 360 detail.
Client 360 Detail

6 tabs: Overview, Trips, Conversations, Preferences, Points, Activity. Household members sidebar.
Client Onboarding

Progressive disclosure form. Paste a call transcript and AI extracts client preferences automatically.

Trip Management

-- 7-stage pipeline from Lead to Archived
Trip Pipeline (Kanban)

7 columns: Lead, Discovery, Planning, Proposal, Booking, Active, Archived. Filter by agent and date range.
Trip Detail

5 tabs: Overview (AI summary + itinerary), Bookings (component cards), Travelers, Tasks, Activity log.

Operations

-- Manager overview, tasks, alerts, configuration
Dashboard

7 KPI cards, SLA health bar, pipeline summary, team workload table, recent activity feed. Manager home screen.
Task List

Priority-striped table with stats bar. Click any row to open slide-over detail panel with activity log.
Maestro AI

Persistent AI chat threads. 2-pane browser: filter by context (trips, clients, conversations, global), search, star, archive. Threads linked from trip detail, client detail, and inbox panels.
Flight Alerts

Alert monitoring table: route, cabin class, points threshold, trigger history. Points-based award flight tracking.
Settings

5 tabs: Team management, SLA configuration, Channels, Appearance (dark/light), Keyboard shortcuts. Includes AI Assistant settings section and Ctrl+J shortcut reference.
